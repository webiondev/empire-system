The module homeworks, example questions and past papers usually prepare you for the style of questions you will get in the examinations. Use these to help you revise and practice.

Some lecturers put homework, assignments and past papers (some with solutions) on module materials section. These are very useful, you can ‘have a go’ then when you are stuck look at the answer and hints and try again.

The examinations & tests are used to check that you have learned the material you have been taught and to see if you can apply it to the subject you are studying. If you are struggling to revise the work, you are usually allowed to ask for help from lecturers.

If things go wrong you can re-sit an examination. Examination date will be set by your lecturer but then the mark is capped at 40%. If you are ill or, for example, have an accident during the usual examination period AND you have a medical note then the re-sit can be treated as a ‘first sit’.

Usually you have to pass all modules in any year to be allowed to progress to the next year. Sometimes because of ill health or difficulty adjusting to University life students need to finish off, or repeat a year, before they can start the next year of the course. This is a nuisance but it is not the end of the world.

You will be notified about the exact examination or test date weeks before the actual exam or midterm test. You can check the status of your examinations or test on the Examination or Test tab of Tests & Examination section, on the left sidebar menu.

Once examination or test is created, it will be available on the Available Examinations list as shown on the image above. If you ready to take your examination, you need to click Start button across from the module title and you will be redirected to the examination or test page.

Examination page include information about test or examination such as: type, duration, code field. When the code will be entered, test or examination questions and tasks will be displayed and timer will be started. Please note that once you enter the code and click Start button the process will be irreversible.

Upon completion the test and multiple choice exam results will be displayed for you. For other type of examinations, the results will be out after lecturers check.
