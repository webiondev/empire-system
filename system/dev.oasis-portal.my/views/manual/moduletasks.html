Module Tasks is one of the core aspects of your educational process. In order to evaluate the level of your knowledges and for you to pass the course module, you will be given different tasks.

The current task list is located in a Course Details section of the menu, under the Tasks tab. "My Tasks" displays the list of the tasks to be completed before the task due date.

In order for you to download the task, first navigate to the action column of the list and click on Download Task button. Once the task is downloaded, open the file and follow the steps for completing your work.

Upon task completion, you need to navigate to the Tasks list page and click Upload Task button, then select a file for the task upload. Thereafter, your work will be evaluated by the lecturer, and results will be published.

The task list widget is also available for you on the dashboard page.
